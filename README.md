# Crystal Spa Website - New Design

Modern, clean website for Crystal Spa based on Figma redesign.

## Setup

### Required Assets

Add these files to the `assets/` directory:

1. **logo.svg** - Crystal Spa logo with moon/stars icon (export from Figma)
2. **hero-massage.jpg** - Hero section massage image (the rounded image from Figma)

### Color Palette

- **Primary Teal**: `#1A4D4F` (buttons, logo, footer)
- **Teal Light**: `#2A6365` (hover states)
- **Gray Background**: `#F5F5F5`

### Typography

- **Font**: Inter (loaded from Google Fonts)
- **Weights**: 300, 400, 500, 600, 700

## Pages Structure

- `index.html` - Homepage (completed)
- `services.html` - Services page (to be created)
- `about.html` - About Us page (to be created)
- `contact.html` - Contact page (to be created)
